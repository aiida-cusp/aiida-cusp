********
Commands
********

The AiiDA package comes with a useful command line interface accessible through the ``verdi`` command.
To simplify certain tasks the default ``verdi`` command line interface is extended with additional commands introduced by this plugin.
In this section all commands added by the plugin and their usage are documented.
